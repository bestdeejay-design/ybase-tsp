# Yell Parser

Автоматизированная система сбора, нормализации и каталогизации данных о заведениях (рестораны, кафе, магазины, услуги) из различных источников.

## Особенности
- Автоматический парсинг данных с yell.ru
- Нормализация и очистка данных
- Каталогизация по категориям
- Веб-интерфейс для просмотра данных
- Фильтрация и сортировка заведений

## Технологии
- PHP 8.2
- PostgreSQL 16
- Docker Compose
- HTML/CSS/JavaScript

## Установка
1. Склонируйте репозиторий
2. Запустите 
3. Импортируйте дамп базы данных

## Использование
После запуска система будет доступна по адресу http://localhost:3000

## Структура
- `src/` - исходный код парсера и обработчики данных
- `public/` - веб-скрипты
- `templates/` - HTML шаблоны
- `database/` - SQL скрипты
- `docs/` - документация
- `dumps/` - дампы базы данных
